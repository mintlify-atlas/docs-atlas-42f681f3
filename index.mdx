## Quick Start

Get up and running with ADK Utils in minutes.

<Steps>
  <Step title="Install the package">
    Install ADK Utils and its peer dependencies
    
    <CodeGroup>
    ```bash npm
    npm install @yagolopez/adk-utils @google/adk @google/genai ai ollama
    ```
    ```bash yarn
    yarn add @yagolopez/adk-utils @google/adk @google/genai ai ollama
    ```
    ```bash pnpm
    pnpm add @yagolopez/adk-utils @google/adk @google/genai ai ollama
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Create your first agent">
    Create an ADK agent using a local Ollama model
    
    ```typescript
    import { OllamaModel } from '@yagolopez/adk-utils';
    import { LlmAgent } from '@google/adk';

    const agent = new LlmAgent({
      name: 'my-agent',
      model: new OllamaModel('qwen3:0.6b', 'http://localhost:11434/v1'),
      description: 'A helpful assistant',
      instruction: 'You are a helpful assistant...',
    });
    ```
  </Step>
  
  <Step title="Start building">
    Explore the guides to learn more about integrating ADK Utils into your applications
  </Step>
</Steps>

## Why ADK Utils?

<Note>
Google ADK provides [local model support for Python via litellm](https://google.github.io/adk-docs/agents/models/litellm/#setup), but there's no built-in way to use local Ollama models with TypeScript. ADK Utils fills this gap.
</Note>

  <Card title="TypeScript First" icon="code">
